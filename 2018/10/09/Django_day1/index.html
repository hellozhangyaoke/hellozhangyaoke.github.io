    Django_day1(快速回忆项目架构)

    • Django的安装：
      >>>pip3 install django
    • 查看Django版本：
      >>>django-admin version
>>>2.1.1
    • 新建项目：
      >>>django-admin startproject {项目名称}
      项目目录架构：

      manage.py应用的命令行接口。
      项目名称文件夹下的：
      settings.py为当前项目的配置文件。
      urls.py为当前目录的url配置文件。
      wsgi.py为服务器与django项目连接的配置文件，部署网站的时候要用到，学习则用不上。
    • 新建app：
      Django-admin startapp {app名称}
      app目录：

      init.py 告诉python这是一个包，
      admin.py为管理界面配置文件，
      Models.py 数据模型，
      views.py视图函数。
      Test.py单元测试。
